A Better Way: Making Each Pet Type Special!

In Java, we can create specific classes for each type of pet. Instead of using a String to tell pets apart, we'll give each type its own special class!

Step 1: Creating Our First Pet Type

Let's start with dogs. Create a new file called Dog.java and add this code:

public class Dog extends Pet {
    public Dog(String name, int age) {
        super(name, "dog", age);
    }
}
New Keywords!
  • extends means "Dog is a type of Pet"
  • super calls the Pet constructor

Step 2: Making Dogs Special

Now we can give dogs their own special way of making sounds:

public class Dog extends Pet {
    public Dog(String name, int age) {
        super(name, "dog", age);
    }

    @Override
    public String makeSound() {
        if (getMood().equals("happy")) {
            return "Woof! Woof!";
        }
        return "Woof!";
    }
}
Notice: We marked the method with @Override to tell Java we're replacing the Pet class version. This helps catch spelling mistakes!

Let's Try It Out!

Update your main method to create a Dog instead of a generic Pet:

public static void main(String[] args) {
    Dog buddy = new Dog("Buddy", 3);
    System.out.println(buddy.getName() + " says: " + buddy.makeSound());
    
    buddy.play();  // Makes Buddy happy
    System.out.println(buddy.getName() + " says: " + buddy.makeSound());
}

Adding Special Dog Behaviors! ü¶Æ

Now that we have specific classes for our pets, we can add behaviors that only make sense for certain types. After all, you wouldn't expect a fish to play fetch, right? üéæ

Teaching Dogs New Tricks

Let's add some dog-specific methods to our Dog class:

public class Dog extends Pet {
    private boolean knowsFetch = false;

    // Constructor stays the same
    
    public void trainFetch() {
        knowsFetch = true;
        setMood("happy");
        System.out.println(getName() + " learned how to fetch!");
    }
    
    public void fetch() {
        if (knowsFetch) {
            System.out.println(getName() + " chases the ball and brings it back!");
            setMood("happy");
        } else {
            System.out.println(getName() + " looks confused... Maybe some training is needed!");
        }
    }

    @Override
    public void play() {
        super.play();  // Do the normal pet play first
        
        if (knowsFetch) {
            System.out.println(getName() + " looks at you expectantly, ready to fetch!");
        }
    }
}
Notice: We've also overridden the play() method, but we still want the original Pet behavior too. That's what super.play() does!

Mixing Different Pet Types! üê±üê∂

Our pets now have their own special behaviors - that's great! But in a real pet shop, we need to work with different types of pets together. Let's see how Java helps us do this!

The Magic of Polymorphism ‚ú®

Here's something cool - since both Dog and Cat are types of Pet, we can store them in Pet variables:

// These all work!
Pet pet1 = new Dog("Buddy", 3);
Pet pet2 = new Cat("Whiskers", 5);

// They still make their own special sounds
System.out.println(pet1.makeSound());  // Woof!
System.out.println(pet2.makeSound());  // Meow!
This is Polymorphism: Even though we're using Pet variables, each pet still behaves according to its actual type (Dog or Cat)! üé≠

But Wait... Where Did Our Special Methods Go? ü§î

Try this code:

Pet pet = new Dog("Buddy", 3);
pet.fetch();  // This won't work! ‚ùå
Why Doesn't It Work? When we use a Pet variable, Java only knows about methods that exist in the Pet class. fetch() is a Dog-specific method!

Type Checking and Casting

If we want to use special methods, we need to tell Java what type of pet we really have:

// First, check what type of pet it is
if (pet instanceof Dog) {
    // Convert (cast) to a Dog to use Dog methods
    Dog dog = (Dog) pet;
    dog.fetch();
}

if (pet instanceof Cat) {
    Cat cat = (Cat) pet;
    cat.giveScratchingPost();
}

Making a Simple Pet Display System

Let's create a method that can handle any type of pet:

public static void displayPet(Pet pet) {
    System.out.println("Meet " + pet.getName() + "!");
    System.out.println("Age: " + pet.getAge());
    System.out.println("They say: " + pet.makeSound());
    
    // Check for special abilities
    if (pet instanceof Dog) {
        Dog dog = (Dog) pet;
        System.out.println("Special ability: Can learn to fetch!");
    } else if (pet instanceof Cat) {
        Cat cat = (Cat) pet;
        System.out.println("Special ability: Uses scratching post!");
    }
    System.out.println();  // Empty line for spacing
}
